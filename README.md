# PasswordRemember
* Remember your password
### Setup

**Required Python 3.9**
 1. execute in cmd:
 > pip install pyperclip
 2. edit services and passwords in main.py 
 > psw = {
    'servicename': 'password',
    'servicename1': 'password',
    'servicename2': 'password',
  }
 
 3. Remember to change the master password (line 16)!
